Once you have downloaded/cloned the repo and opened it, you will find the following structure:

```
|-- hermes-dashboard
	|--assets
		|--css
		|--fonts
		|--img
		|--js
			|--core
			|--plugins
			|--demo.js
			|--hd.js
		|--sass
	|--docs
	|--examples

```
