Hermes Dashboard

Hermes Dashboard is a beautiful admin template built over Bootstrap 4 that allows you to create powerful and beautiful dashboards.

Currently v{{ site.current_version }}

We have redesigned all the usual components in Bootstrap to make it look flat, minimalist and easy to use. You can safely use it on your existing or new Bootstrap project. No line of code from Bootstrap 4 was changed, so you don't have to worry about undesired effect in your work.

To jump start your project, you can use our start-up examples/template.html where all the files are already included and ready to use. Or you can do this manually as we provide all the necessary CSS and JS resources. So, to immediately change or get started with our design, include the assets/css/hermes-dashboard.css in your HTML template.

The core JavaScript is contained in assets/js/hd.js. Some functions are called from assets/js/demo.js because they are presented only for demo purpose.
